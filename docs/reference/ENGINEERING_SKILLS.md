# Engineering Skills

Lifeline uses repo-native engineering skills to encode stable governance and operating rules before adding higher-level automation layers.

## Phase order

1. skills
2. agents
3. teams
4. workflows

## Current skills

- [documentation-governance](../../.github/skills/documentation-governance.md)
- [cicd-governance](../../.github/skills/cicd-governance.md)

## Skill intent

- skills capture repo-specific judgment, safeguards, routing rules, and review heuristics
- agents use these skills to execute domain workflows more consistently
- teams build on those agent-level responsibilities
- workflows build on the same rules rather than inventing a separate system
